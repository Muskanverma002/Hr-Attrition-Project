# HR Analytics Dashboard with Python, SQL, ML & Power BI

## 📌 Project Overview
This project analyzes HR data to understand employee performance, attrition, and retention patterns.  
It uses **Python, SQL, Machine Learning, and Power BI** to provide actionable insights for HR teams.

---

## 🛠️ Tools & Technologies
- **Python**: Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn  
- **SQL**: PostgreSQL for querying and cleaning data  
- **Machine Learning**: Predictive models for attrition analysis  
- **Power BI**: Interactive dashboards (attrition, satisfaction, performance metrics)  

---

## 📂 Project Structure
- `data/` → Sample HR dataset  
- `sql/` → SQL queries & schema  
- `notebooks/` → Jupyter notebooks (Python ML models & EDA)  
- `dashboard/` → Power BI dashboard file (`HR_Analytics.pbix`)  
- `src/` → Python scripts for preprocessing & ML models
